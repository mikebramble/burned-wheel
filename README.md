# Burning Wheel Character Sheet

This repository hosts a custom, lightweight, and interactive digital character sheet for the Burning Wheel TTRPG.

## What is this?

This tool provides a persistent, browser-based character sheet that allows you to manage stats, skills, traits, and physical tolerances directly within your web browser.

## How to use it

### Access the Sheet
Navigate to the live page hosted on GitHub Pages

### Edit your Character
Simply click into any text box, checkbox, or number selector to input your character's information.

### Local Saving
Your browser will automatically save your progress. You can close the tab or turn off your computer, and your data will remain waiting for you when you return.

### Manage Rows
Use the **"+ Add Row"** buttons to expand **Skills**, **Traits**, **Relationships**, or **Contacts** as your character grows throughout a campaign.

Use the **"×"** button on the far right of any dynamic row to delete it if needed.

### Export/Import
Use the **"Export to File"** button to download a `.json` backup of your character.

You can use **"Import File"** to load that same backup onto any other device or browser.

### Print to PDF
Click the **"Print / Save as PDF"** button to generate a physical or digital snapshot of your current character sheet.
